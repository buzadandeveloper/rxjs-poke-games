# React + RxJS Poke Games Learning Project

A learning project for mastering **RxJS** and understanding how reactive operators work. Built with React, TypeScript,
and RxJS to practice async logic, observable streams, and operator composition.

## Purpose

This project is created to learn:

- RxJS fundamentals and observable patterns
- How to use operators (map, switchMap, forkJoin,combineLatestWith etc.)
- How to compose operators for powerful reactive logic
- State management with observables in React

## Quick Start

### Prerequisites

- **Bun** (or Node.js)

### Installation & Running

```bash
# Install dependencies
bun install

# Start development server
bun dev
```

The app will be available at `http://localhost:5173`

## Tech Stack

- React 19
- TypeScript
- RxJS 7.8
- React Router 8
- Vite
- Tailwind CSS + DaisyUI
